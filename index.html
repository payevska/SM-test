Example task #3
Js можем подключить в html файле несколькими способами: <head> и <body> можно встроить: <script> код js </script> или <script type="text/javascript" src="script.js"></script> - ссылка на внешний файл. Лучше подключать ссылку на внешний файл в <body> в конце тела страницы. Во время загрузки браузер загружает сначала все, что находится в <head> странице, потом переходит к <body> - пользователь будет видеть содержимое - тело страницы во время загдузки браузера. Внешний файл можно включать в несколько страниц и его проще сопровождать.
CSS можем подключить в html файле несколькими способами: в <head> можно встроить: <style> код js </style&g; или <link rel="stylesheet" href="style.css"> - ссылка на внешний файл. В <body> стили записывать в тегах. Лучше подключать ссылку на внешний файл в <head> после тега <title>. Во внешнем файле может находиться большой код, если такой код разместить в теге <head> - это усложнит чтение и работу с html файлом. В <body> стили можно записывать в тегах, но увеличится общий объем файлов и время загрузки браузера увеличится так же.

Example task #5
Блок, имеющий свойство visibility: hidden или opacity:0 невидим, но он занимает тоже место и размеры, с visibility: hidden мы не можем взаимодействовать с блоком, его свойство наследуется и ребенок может сменить hidden на visible.
overflow: hidden, в сравнении с visibility: hidden и opacity:0, блок можно удалить из потока контента с помощью position: absolute и отрицательного значения top или left, скрывает содержание блока, выходящее за его пределы.

null показывает, что переменная «пустая»: var a = null; if (a === null) { return 0; }. console.log(typeof a); // object
undefind обычно означает, что переменная объявлена, но не имеет присвоенного значения: var a; if (a === undefined) { // эти инструкции будут выполнены }. console.log(typeof a); // undefined
